# AGENTS.md — sphinx-no-pragma

## Project overview

sphinx-no-pragma is a Python Sphinx extension that strips pragma comments (e.g. `# type: ignore`, `# noqa`, `# pragma: no cover`) from source code included in documentation via `.. literalinclude::`.

- **Primary language**: Python 3.10+
- **Package type**: Single-file module (`sphinx_no_pragma.py`, configured as `py-modules` in `pyproject.toml`)
- **Package manager**: `uv`
- **Documentation**: Sphinx with reStructuredText, hosted on Read the Docs

## Architecture invariants

- The project is a **single-file module** — there is no `sphinx_no_pragma/` package directory. All logic lives in `sphinx_no_pragma.py`.
- The `setup()` function registers `NoPragmaLiteralInclude` as an override of Sphinx's `literalinclude` directive and adds two config values: `ignore_comments_endings` and `user_ignore_comments_endings`.
- `DEFAULT_IGNORE_COMMENTS_ENDINGS` is the canonical list of pragma markers to strip.
- The package is **not** a namespace package or a standard `py-modules` package with submodules.
